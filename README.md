## My favorite projects

### [42sh](https://github.com/Thegeekeuhhhhhhh/42sh)

> _Just a shell interpreter, not that complex :o_

Implemented in C, covers piping, redirections, builtins, and all the POSIX-compliant things that make you question your life choices.

---

### [GeoMaster](https://github.com/Thegeekeuhhhhhhh/GeoMaster)

> _Learn everything about geography, simple._

A cross-platform app built in Dart / Flutter. I use it daily (no jokes)

Try it out: https://geo-master.thegeekeuhhhhhhh.dev/

---

### [OCR](https://github.com/Thegeekeuhhhhhhh/OCR)

> _Optical Character Recognition engine to solve sudokus._

Built in C for EPITA S3. Neural network from scratch, no ML libraries allowed. Everything made by hand (except the C compiler).

---

### [PING](https://github.com/Thegeekeuhhhhhhh/PING)

> _Snow removal management app for Montreal. (Yes)_

A Java / JavaScript / React application that simulates and optimizes snow removal logistics across a city.

---

### [SpotiFree](https://github.com/Thegeekeuhhhhhhh/spotifree)

> _A desktop app to listen to music._

Built in JavaScript. Similar to Spotify, but, free ;)

---

### [DroXyd](https://github.com/Thegeekeuhhhhhhh/DroXyd)

> _A web browser, better than chrome_

Written in Rust for EPITA S4. Parses HTML, renders pages, and crawls Internet to store websites, and reminds you that browsers are deceptively hard.
